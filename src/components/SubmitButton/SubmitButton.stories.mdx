## How To Use

Bind your SubmitButton to a useState Hook

```javascript
const [submitted, setSubmitted] = useState(false)
const onSubmit = (e)=>{
    e.preventDefault()
    setSubmitted(true)
}
return (
    <form onSubmit={event => onSubmit(event)}>
        <SubmitButton submitted={submitted} {...args} />
    </form>
)
```
