# Examples for Storybook with React

### Install Dependencies
    npm install

### Start Storybook
    npm run storybook

### Run Tests
    npm run test
    npm run test-coverage

### Update Coverage Badge ![code-coverage-badge](https://img.shields.io/badge/coverage-100%25-brightgreen)
    npm run update-badge
